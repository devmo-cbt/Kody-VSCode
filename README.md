# Kody

**CB&T DevAgent** — a Local AI Agentic Agent for the Tiger Team.

Kody can handle complex software development tasks step-by-step with tools that let it create and edit files, explore large projects, use the browser, and execute terminal commands (with your permission).

## Features

- File creation, editing, and management
- Terminal command execution
- Browser automation
- Code analysis and refactoring
- SQL Server expertise (star schema, SCD Type 2)
- MCP server integration

## Setup

Kody connects to the internal Kody API on the Mac Studio coordinator. No additional configuration needed — the endpoint is pre-configured.

**Model:** `kody-local` (default) routes to on-prem GPU infrastructure.
